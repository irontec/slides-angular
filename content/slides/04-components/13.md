## Proyección - ng-content

Este concepto es útil a la hora de diseñar la jerarquía de componentes de nuestra aplicación:

Partimos de un componente padre:
```ts
@Component({
  selector: 'app-item',
  template: `<li>
    <a [href]="link">
        <ng-content></ng-content>
    </a>
    </li>`,
})
export class ListComponent {
    @Input() link:string;
}
```
