## Property binding

Es posible utilizar [corchetes] para asignar propiedades on atributos del template.

```typescript
@Component({
  selector: 'app-list-items',
  template: `
    <button [disabled]="buttonDisabled">Bot√≥n ({{ buttonDisabled }})</button>
  `
})
export class ListItemsComponent{
  buttonDisabled = true;
  constructor() { }
}
```
