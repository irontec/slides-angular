## Directiva de estructura *ngFor

Directiva que permite iterar un atributo de un componente:
```ts
@Component({
  selector: 'app-list-items',
})
export class ListItemsComponent{
  items: Items[];
  / ... /
}
```

```html
<div *ngFor="let item of items; trackBy: trackById">
  {{item.name}}
</div>
```
