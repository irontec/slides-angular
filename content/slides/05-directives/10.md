## Directivas de atributo - ngModel

```ts
@Component({
  selector: 'app-value',
})
export class ListItemsComponent{
  valor: string = "por defecto"
  setValue(v:string) {
      this.valor = v;
  }
}
```
```html
<input [(ngModel)]="valor">
```
```html
<input [ngModel]="valor" (ngModelChange)="setValue($event)"/>
```
