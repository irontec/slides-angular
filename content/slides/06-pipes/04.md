## Pipes - custom (I)

Podemos crear nuestros propios pipes (**Angular CLI!**):

```shell
$ ng generate pipe emoticon 
CREATE src/app/emoticon.pipe.spec.ts (195 bytes)
CREATE src/app/emoticon.pipe.ts (205 bytes)
UPDATE src/app/app.module.ts (765 bytes)
```
